

#### space-constant [<i class='fa fa-github' aria-hidden='true'></i>](https://github.com/tangrams/blocks/tree/gh-pages//space/constant.yaml)
This provides the following blocks:

- **global**:
 + `vec2 getConstantCoords () `

Import it using:

```yaml
import:
    - https://tangrams.github.io/blocks/space/constant.yaml
```




If you want to import this block with dependences included try this:

```yaml
import:
    - https://tangrams.github.io/blocks/space/constant-full.yaml
```
