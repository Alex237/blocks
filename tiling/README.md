

### [tiling-brick](https://github.com/tangrams/blocks/blob/gh-pages/tiling/brick.yaml)

This provides the following blocks:

- **global**:
 + `vec2 brick (vec2 st, float zoom) `

Import it using:

```yaml
import:
    - http://tangrams.github.io/blocks/./tiling./tiling/brick.yaml
```
